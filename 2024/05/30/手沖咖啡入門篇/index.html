手沖咖啡入門篇

手沖咖啡是一種流行的手工沖煮方法，以其細膩的風味和可控的沖煮過程深受咖啡愛好者喜愛。手沖咖啡的水法和技巧是沖煮過程中的關鍵部分，以下是詳細的說明和步驟：

所需器具和材料

  1. 手沖壺
  2. 手沖咖啡濾杯（如V60、Chemex）
  3. 咖啡濾紙
  4. 咖啡豆
  5. 磨豆機
  6. 計量器具（如電子秤）
  7. 溫度計（選用）
  8. 計時器
  9. 純淨水

步驟和技巧

準備工作

  1. 稱量咖啡豆：
    • 根據你的需求，一般比例為1:15到1:17，即1克咖啡豆對應15到17克水。
    • 例如，20克咖啡豆配300克水。
  2. 磨豆：
    • 使用磨豆機將咖啡豆磨成適中均勻的顆粒，手沖咖啡通常選擇中粗研磨。
  3. 準備水：
    • 使用純淨水，加熱至90-96°C之間。溫度太高會過度萃取，太低則可能萃取不足。
  4. 濕潤濾紙：
    • 把濾紙放入手沖咖啡濾杯，然後用熱水徹底沖洗濾紙，這樣可以去除紙味並預熱濾杯。

手沖技巧

  1. 悶蒸（Blooming）：
    • 將磨好的咖啡粉倒入濾杯，輕輕搖動濾杯使粉末均勻平整。
    • 倒入約2倍於咖啡粉重量的熱水（例如20克咖啡豆，倒入40克水），讓水充分濕潤咖啡粉，然後靜置約30秒，這段時間稱為悶蒸，目的是讓二氧化碳釋放出來，便於後續萃取。
  2. 螺旋注水法：
    • 使用手沖壺以小水流從中心開始，按螺旋方式慢慢向外注水，直至達到所需水量的一半左右。
    • 注水時保持手沖壺距離濾杯表面約1-2厘米，確保水流均勻且穩定。
  3. 多段注水：
    分段注水是關鍵，通常將剩餘水量分2到3次加入，每次注水間隔約30秒，這樣可以均勻萃取。
    • 每次注水時，仍然保持螺旋方式，確保所有咖啡粉均勻接觸到水。
  4. 最後注水：
    • 最後一次注水時，可以將濾杯內的咖啡粉輕輕搖動，使其表面平整，這有助於最後的均勻萃取。

完成沖煮

  1. 滴濾完成：
    • 當所有水注完後，讓咖啡繼續滴濾直至水完全流過咖啡粉，整個過程應在3到4分鐘內完成。
    • 如果滴濾時間過長或過短，可以調整下次的磨豆粗細或水量。
  2. 享受咖啡：
    • 移除濾杯，將咖啡倒入預熱的杯子中，靜待片刻即可享用。

水法和技巧總結

  1. 水質：選用純淨水，以確保不會影響咖啡風味。
  2. 水溫：保持在90-96°C之間，避免過度或不足萃取。
  3. 悶蒸：確保咖啡粉在萃取前充分濕潤並釋放二氧化碳。
  4. 螺旋注水：保持均勻穩定的水流，確保所有咖啡粉均勻接觸到水。
  5. 分段注水：控制注水的速度和次數，避免一次性注水過多。
  6. 總時間控制：整個手沖過程應在3到4分鐘內完成，根據時間調整磨豆粗細或注水量。

掌握這些水法和技巧，可以幫助你沖煮出風味豐富、口感均衡的手沖咖啡。每次沖煮都是一次實驗，根據實際效果不斷調整，找到最適合自己的沖煮方法。
